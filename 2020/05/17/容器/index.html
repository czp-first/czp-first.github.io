  1. docker容器的网络访问

    指定映射（docker会自动添加一条iptables规则来实现端口映射）

    -p hostPort:containerPort

    -p ip:hostPort:containerPort

    -p ip::containerPort(随机端口)

    -p hostPort:containerPort/udp

    -p 81:80 -p 443:443

    随机映射

    docker run -p 80(随机端口:32768-60999)

    允许多端口映射
