一台电脑配置多个github账号的ssh

Posted on 2020-05-24 In git Views:

前言
工作有一个github账号，自己有一个github账号，都需要配置ssh，怎么办？

  1. ssh-keygen -t rsa -C “邮箱1”

    将id_rsa_self.pub内容复制到账号1中

  2. ssh-keygen -t rsa -C “邮箱2”

    将id_rsa_work.pub内容复制到账号2中

  3. vim ~/.ssh/config

Host github.com
HostName github.com
User self
IdentityFile ~/.ssh/id_rsa_self

Host github_work
HostName github.com
User work
IdentityFile ~/.ssh/id_rsa_work
